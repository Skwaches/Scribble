# Scribble
This is a simple neural network that recognises numbers drawn on a 28 by 28 pixel grid.
It is inspired by this series by [3blue1brown](https://www.youtube.com/playlist?list=PLZHQObOWTQDNU6R1_67000Dx_ZCJB-3pi).

## Dependencies 
`SDL3`

## Data set
All the data used is form the [MNIST Dataset](http://yann.lecun.com/exdb/mnist/)
